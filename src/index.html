Drones 101

Welcome to the Drone 101 course!

This course is an introductions to drones and all that they are capable of. In the middle of this page you will see some block codes. This is where you will find different blocks that will control your drone to the right. As you learn more about drones you will have challenges that reflect what drones can do in real life. Complete the challenges to move on to the next lesson.

Key Components:

  • Flight Controller: The "brain" of the drone, responsible for processing inputs and controlling the motors.
  • Motors & Propellers: The components that generate thrust to lift and move the drone.
  • Battery: The power source for the drone.
  • Sensors: Components like GPS, gyroscopes, and accelerometers that provide data about the drone's position and orientation.

In this application, you will use Blockly, a visual programming language, to create sequences of commands for a simulated drone. Watch the simulation to see your code in action!
